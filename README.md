# israelseas

Homeassistant intergration for Israel beaches weather monitoring.

## Configuration example

``` 
- platform: israelseas
  scan_interval: 00:01:00  
  beaches:
      - Northern Coast
      - Southern Coast
      - Sea of Galilee
      - Gulf of Elat
```


Data collected from [ims.data.gov.il](https://ims.data.gov.il/).
